# Gregorius Peitra — Academic Portfolio

Academic portfolio website for research profile and CV, focused on Digital & Sustainable Customer Experience.

## Stack

- Pure HTML/CSS/JS — no build step required
- IBM Plex Sans + IBM Plex Serif (Google Fonts)
- Deploys directly to Vercel or GitHub Pages

## Development

Simply open `index.html` in a browser, or serve with any static server:

```bash
npx serve .
```

## Deploy to Vercel

1. Push this repo to GitHub
2. Import project in [vercel.com](https://vercel.com)
3. Deploy — zero config needed

## Pages

- `/index.html` — Home / Hero
- `/about.html` — Research profile
- `/roadmap.html` — 5-year research plan
- `/projects.html` — Research projects
- `/cv.html` — Academic CV
- `/contact.html` — Contact information
